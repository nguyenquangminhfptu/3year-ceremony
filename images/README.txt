HƯỚNG DẪN SỬ DỤNG FOLDER ẢNH KỶ NIỆM
=====================================

1. Đặt ảnh kỷ niệm của bạn vào folder này
2. Đặt tên ảnh theo format: photo1.jpg, photo2.jpg, photo3.jpg...
   (Hỗ trợ: .jpg, .jpeg, .png, .gif, .webp)

3. Hoặc bạn có thể chỉnh sửa file script.js:
   - Tìm dòng: const MEMORY_IMAGE_NAMES = [...]
   - Thêm tên ảnh của bạn vào mảng đó

Ví dụ:
- photo1.jpg
- photo2.jpg
- photo3.jpg
- my_memory.png
- anniversary.jpg

Lưu ý: Nếu không có ảnh, website sẽ tự động dùng ảnh placeholder.
