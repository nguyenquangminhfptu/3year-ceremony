HƯỚNG DẪN THÊM NHẠC
==================

1. Đặt file nhạc vào folder này với tên: background.mp3
   (Hoặc background.ogg, background.wav)

2. Các định dạng hỗ trợ:
   - .mp3 (khuyến nghị - tương thích tốt nhất)
   - .ogg
   - .wav

3. Lưu ý:
   - File nhạc nên có kích thước hợp lý (không quá lớn)
   - Nhạc sẽ tự động lặp lại (loop)
   - Nhạc sẽ bắt đầu phát sau khi nhập password đúng
   - Có thể điều chỉnh âm lượng bằng thanh trượt ở góc dưới bên phải

4. Nếu không có file nhạc, website vẫn hoạt động bình thường (chỉ không có nhạc nền).
